# ideaNinja.io Build System

Release 1.0

This build system uses Nodejs, NPM and Gulp with the following node modules that you need to install previously: 
 - gulp
 - gulp-util
 - gulp-browserify
 - gulp-compass
 - gulp-connect
 - gulp-if
 - gulp-uglify
 - gulp-minify-html
 - gulp-concat
 - path
 - gulp-save
 - gulp-sitemap
 - gulp-ejs
 - gulp-imagemin
 - imagemin-pngquant
 - gulp-robots
 - run-sequence
 - gulp-clean
